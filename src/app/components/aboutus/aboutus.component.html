Application Overview

Frontend: Angular 19

The frontend is built with Angular 19, the latest version of the modern TypeScript framework. It enables the creation of dynamic, component-based single-page applications (SPAs) with robust tooling and excellent performance.

State Management: NgRx

NgRx is used for managing the application state in a reactive and predictable manner using the Redux pattern. It simplifies state transitions through actions, reducers, selectors, and effects—enhancing scalability and maintainability in large applications.

Backend: Java Spring Boot

The backend leverages Java Spring Boot to build secure and scalable RESTful APIs. Spring Boot streamlines application setup and development with auto-configuration, embedded servers, and production-ready features.

Authentication: JWT

JSON Web Token (JWT) is implemented for secure, stateless user authentication. After login, a token is issued to the client, which is sent with each request to validate access and ensure user identity without session storage. To enhance security and protect against client-side vulnerabilities like XSS (Cross-Site Scripting), the JWT is stored in HTTP-only cookies. These cookies are inaccessible to JavaScript, reducing the risk of token theft through malicious scripts. Additionally, setting the Secure flag ensures the cookie is only transmitted over HTTPS, and using the SameSite=Strict or Lax attribute helps prevent CSRF (Cross-Site Request Forgery) attacks. This approach provides a balance between the security of traditional server sessions and the stateless nature of JWT authentication

Database: PostgreSQL

PostgreSQL is used for managing structured data with reliability and advanced querying capabilities. As an open-source RDBMS, it supports complex transactions, JSON data types, and high-performance indexing.

Charting: Chart.js with ng2-charts

This application uses Chart.js along with the ng2-charts Angular wrapper for interactive data visualization. Chart.js provides flexible, responsive, and customizable chart types such as bar, line, pie, radar, and more. The ng2-charts library seamlessly integrates Chart.js into Angular applications using reusable components, binding, and Angular change detection.

Project Summary

This full-stack web application combines Angular 19 for a modern and responsive UI, Java Spring Boot for a powerful backend, and PostgreSQL for robust and scalable data management. JWT Authentication ensures secure access control, while NgRx provides a reactive and centralized state management solution.

For rich data visualization, the application leverages Chart.js with the ng2-charts wrapper. This allows the seamless integration of dynamic and responsive charts such as bar, line, pie, and radar charts into Angular components, offering users clear and interactive insights into the application data.

The overall architecture ensures performance, security, maintainability, and scalability—ideal for enterprise-level application development.
